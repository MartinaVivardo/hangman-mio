## Hangman
-Un juego clásico del **Ahorcado** desarrollado con **HTML, CSS y JavaScript**.  
-El objetivo es adivinar la palabra secreta antes de que se complete el dibujo del ahorcado.
-Demo: https://hangman-9qpi.onrender.com
# 🕹️ Cómo jugar

1. El juego elige una palabra secreta al azar.  
2. Escribe una letra en el campo de texto o presiona **Enter** para intentar adivinar.  
3. Si la letra está en la palabra, se revelará en su posición.  
4. Si no está, perderás un intento y se dibujará una parte del ahorcado.  
5. Ganas si completas la palabra antes de que se acaben los **6 intentos**.  

## 💻 Tecnologías utilizadas

- **HTML5** → estructura del juego  
- **CSS3** → estilos y diseño visual  
- **JavaScript (ES6)** → lógica del juego
Hangman / Ahorcado

Hangman (Ahorcado) es un juego clásico de adivinanzas de palabras.
El objetivo es descubrir una palabra secreta antes de quedarte sin intentos.

🧩 ¿Cómo se juega?

Un jugador (o el sistema) elige una palabra secreta.

El jugador intenta adivinar la palabra letra por letra.

Si la letra está en la palabra, se revela en su posición.

Si la letra no está, se resta un intento.

El juego termina cuando:

El jugador adivina toda la palabra ✅ (gana), o

Se acaban los intentos ❌ (pierde).

💻 Características del proyecto

Desarrollado con HTML, CSS y JavaScript puro.

Interfaz visual simple y moderna.

Teclado en pantalla y soporte para letras con acentos y Ñ.

Opción para ingresar tu propia palabra secreta (modo 2 jugadores).

Compatible con teclado físico.

Contador de intentos restantes y letras incorrectas.

Preparado para publicarse fácilmente con GitHub Pages.
